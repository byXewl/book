用户定义的快捷键：对应的keybindings.json文件内容

常见系统定义的快捷键：
```
F2，改文件名字

shift+回车 ，下一行新开始

ctrl + 逗号, 进入设置

shift+esc ，打开终端，点击右上角切换面板也可以

ctrl+1，关闭或打开左边资源管理器目录

ctrl+shift+f，全文件搜索内容字符

alt + -> / ctrl+tab，切换编辑器栏

alt+鼠标左键   多选中


ctrl+f 本页搜索，可以进行正则表达式匹配和替换字符，输入框中alt+enter可以选择全部搜索结果。
```
