

Flutter（官方 `flutter build web`）

默认混淆：默认就启动
命令里已经帮你跑了一遍 **dart2js 的 `--minify`** + **额外的标识符混淆**：

  ```
  flutter build web --release
  ```

  结果：

  * 所有 Dart 标识符（变量、类、方法）被映射成**随机短名**（`_a`, `_b` …）且**每次构建不同**。
  * 字符串**不加密**，但**树摇+死码消除**后只剩必要字符。
  * **不会**生成 source-map（除非显式 `--source-maps`），于是浏览器里几乎**看不到原名称**。

* **强度**：介于 **Terser 压缩** 与 **OB 混淆** 之间——**比前端框架默认强，但不到 OB 那种控制流 flatten/字符串加密级别**。
