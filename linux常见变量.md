

###  Linux 中可以直接使用的常见变量（无需定义）：



| 变量名         | 含义说明               |
| :---------- | :----------------- |
| `$PWD`      | 当前工作目录  一般同/proc/self/cwd |
| `$HOME`     | 当前用户的主目录 同~           |
| `$USER`     | 当前用户名              |
| `$SHELL`    | 当前使用的 shell 路径     |
| `$PATH`     | 可执行程序的搜索路径         |
| `$HOSTNAME` | 当前主机名              |
| `$UID`      | 当前用户的 UID（数字）      |
| `$RANDOM`   | 随机数（0–32767）       |
| `$?`        | 上一个命令的退出状态码        |
| `$$`        | 当前 shell 的进程号（PID） |

$IFS 分隔命令

localhost 是127.0.0.1
主要在curl 、ping中用。
