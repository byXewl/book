## **1. 安装Nmap：**
```
apt update && apt install nmap
windows：官网安装图形化namp
```
^
## **2. 常用命令：**
```
nmap -v      版本7.9
     ip/ip段    目标
    -v            看进度日志
    -A            详细扫描

    -F           快速对常见端口扫描

    -T            指定时间模板
    -T3           默认模板，根据目标反应自动调整时间
    -T0           非常慢，用于IDS(waf 网络监控系统) 躲避
    -T1    -T2  慢速
    -T4    -T5  快速
    
    -sS            TCP连接扫描，三次挥手扫描，SYN扫描
    -sU            UDP扫描       理论上可以躲避waf
    
    -p  1-65535       指定目的端口范围扫描

    -Pn  / -PO     关闭主机发现，直接对目标进行端口扫描。无PING扫描，默认为有PING扫描，但是有的站点关闭的可Ping，Windows默认关闭Ping。
    -sP            只进行Ping。
   
    -O            识别操作系统
    -sV           版本探测:目标主机上正在运行的服务的确切版本号

    -oN scan_results.txt     扫描结果保存到文件

     --script=脚本名
     -sC=脚本名
```
^
## **3. 返回状态**
```
端口是否开启：
关闭 closed  （红色）
开启 open    （绿色）
无法判断 fitered （被防火墙拦截过滤，无法判断端口是否开启）
```
^
## **4. 常用模板**
```
指定端口范围有进度日志的扫：
nmap -p 80-6000  -A -v 81.70.185.137

快速扫描常见端口：
nmap -T4 -F 47.109.58.205

详细探测操作系统版本：
nmap  -O 192.168.0.2

全端口扫描：
nmap -p 1-65535 -T4 -A -v 192.168.4.202

nmap -sS -p- -O 192.168.1.3
-sS 半连接扫描减少流量信息， -p- 扫描全端口  -O 扫描系统信息
```
