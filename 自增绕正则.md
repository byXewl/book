有如下
```
"A"++ ==> "B"
"B"++ ==> "C"
```
也就是说，如果我们能够得到"A"，那么我们就能通过自增自减，得到所有的字母。那么问题就转化为怎么得到一个字符"A"。
在PHP中如果强制连接数组和字符串的话，数组将被转换成字符串，其值为"Array"。
再取这个字符串的第一个字母，就可以获得"A"。
```
<?php
$a = ''.[];
var_dump($a);
```
