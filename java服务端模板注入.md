SSTI:Server Side Template Injection，即服务端模板注入。
在Web开发中，为了使用户界面与业务数据分离，提升开发效率，因此会使用模板引擎来生成一个标准的HTML文档用来数据的展示。
常见于前后端不分离的项目。
