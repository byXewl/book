## **逻辑混淆jsfuck**
又称为jsfuck混淆，可以通过工具一键混淆和还原。

`!+[]` 的计算过程如下：
1. `[]` 被转换为数值 `0`。
2. `+[]` 因此是 `0`。
3. `!0` 转换为 `true`。
```
!+[]
true

!+[]+!+[]
2

[!+[]+!+[]+!+[]+!+[]+!+[]]+[!+[]+!+[]]
'22'

![]
false

!+[]
true

false+[]
'false'

!+[]+[]
'true'

(!+[]+[])[!+[]+!+[]]
'u'
```

^
对象、函数名混淆
```
_0x101dae['data']['result']
直接将整个复制到控制台打印
复制前面部分到控制台打印就是原码
```
