一个压缩包里有很多文件
需要用脚本批量提取

^
## **内容提取**
每个文件中有内容
```
提取内容为一个文件中，无空格换行拼接。

提取内容为一个文件中，有换行的拼接。（字典、多重base编码）
```

^
每个文件中内容一样，或无内容
```
提取每个文件创建时间的最两位，转为ascii字符
提取每个文件修改时间的最两位，转为ascii字符
提取每个文件创建时间的间隔，转为ascii字符

提取文件名为一个文件中，无空格换行拼接。（某种编码）
提取文件名为一个文件中，有换行的拼接。（字典）
```

^
## **内容作用**
提取后内容可能得作用
```
就是flag字符的编码

某种特殊的编码

某种文件的16进制

爆破密码的字典
```
